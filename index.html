Description :

This is a console-based Tic-Tac-Toe game developed in Java using Object-Oriented Programming (OOP) principles.The game supports two modes:

  • player vs player
  • player vs computer(machine)

key features :

  • OOP-Based Structure:Separate classes for Game, Player, Board, and ComputerPlayer.Use of encapsulation, inheritance, and polymorphism.
  • Multiple Game Modes:Option to play with another human player or against the computer.
  • Simple Console UI:Board is displayed in the console.Input is taken using Scanner for positions.
  • Input Validation:Prevents overwriting already filled cells.Ensures input is within valid range.
  • Game Status Checker:Automatically checks for win, draw, or ongoing game after every move.
  • Basic AI for Machine:The machine selects moves strategically or randomly (can be upgraded to use Minimax).
  • Replay Option:Allows players to play again without restarting the application.

INSTAGRAM CLONE

click here to view project

Description :

The Instagram Clone is a simplified social media web application built using Java technologies such as JSP (JavaServer Pages), Servlets, MySQL, and styled with CSS and JavaScript for interactivity. It replicates the core functionalities of Instagram, allowing users to register, log in, share posts (photos/videos), interact with others through likes and follows, and explore user profiles in a dynamic, responsive interface.

key features :

  • User Authentication:Sign up, log in, and log out functionality.Secure session handling.
  • Post Uploading:Upload and display photos or videos.Captions supported.Posts stored with timestamps and linked to users.
  • Post Feed:Dynamic home feed showing all posts from users.Includes profile image, username, caption, media, and like button.
  • Like System:Users can like posts and reels.Like count displayed dynamically.
  • Follow/Unfollow System:Users can follow or unfollow others.Follower and following counts updated in real-time.
  • User Profiles:Each user has a profile page showing their posts and bio.View other users’ profiles by clicking on usernames.
  • Reels Feature:Short video format similar to Instagram Reels.Displayed in a scrollable view.
  • Media Storage with Cloudinary:Uploaded images and videos are stored using Cloudinary integration.
  • Bookmark/Saved Posts:Users can save posts and reels to view later.
  • Clean MVC Architecture:Separates concerns into Models (Java classes), Views (JSP), and Controllers (Servlets).
